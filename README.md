### calculator
- [x] 사칙연산 기능 구현
- [x] 사칙연산 예외 (0 나눗셈) 처리

- [x] 문자 계산기 기능 구현

  - [x] JUnit5 기반 테스트 작성
  - [x] AssertJ 기반 테스트 작성 (리팩터링)
